# Quietz

[Quietz](https://quietz.netlify.app) is my private playground for Nuxt.js and Bootstrap 5.

## Features

- Using Nuxt.js and Bootstrap 5
- Full static generation
- Using PurgeCSS
- Using CSS Modules
- Not Using [BootstrapVue](https://bootstrap-vue.org/)
